# FriendsGacha バランス仕様案

## アップグレード上限

  項目                 上限 備考
  ------------------ ------ ----------------------------
  製造速度             Lv50 生産間隔は最短0.1秒まで
  一度に生産する量     Lv20 最終100万個/回
  並列生産ライン       20本 UI・バランスを考慮
  タップの効果         Lv10 最終で残り時間を1/11に短縮

## アップグレードコスト

  項目                 初期コスト    倍率   上限
  ------------------ ------------ ------- ------
  製造速度                 15マメ   ×1.35   Lv50
  一度に生産する量         25マメ    ×2.0   Lv20
  並列生産ライン          150マメ    ×2.0   20本
  タップの効果             20マメ    ×1.8   Lv10

## 通貨

### マメ

-   所持上限：999,999,999

### ナゾマメ

-   所持上限：9,999
-   1億マメで1個交換
-   自動変換ではなく交換ボタンから任意で交換

## ナゾマメガチャ

-   消費：ナゾマメ10個
-   R以上確定
-   SSR確率は通常より少し高めに調整予定（例：R70% / SR25% / SSR5%）

## バランス方針

-   序盤は製造速度でテンポ良く進行
-   中盤は生産量強化が主力
-   終盤は並列ライン増設が最大の伸び
-   タップ強化は補助要素
-   最終目標は1億マメを集めてナゾマメへ交換し、R以上確定ガチャを回すサイクルを目指す
